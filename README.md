# Taller Evaluación de Software

## Descripción
Este repositorio contiene el desarrollo del taller de evaluación de software, donde se analizan conceptos como calidad del software, medición, evaluación, metodologías como Six Sigma y CMMI, y fases de aseguramiento de calidad (SQA).

## Contenido
- Documento del taller en formato PDF
- Respuestas a las preguntas de los videos
